EventBook genel etkinlik duyurularını ve oluşturulan bu etkinliklerin takibini, davetlilerin katılıp katılmama durumunu ve kullanıcıların plan çizelgelerinin gösterildiği bir etkinlik yönetim sistemi uygulamasıdır. Bu proje içerisinde kullanıcılar kolaylıkla üyelik oluşturabilecek, sistemde kayıtlı etkinliklere katılma durumunu belirleyebilecek ve istediği takdirde kendi etkinliğini oluşturup katılan kişileri görüntüleyebilecektir.


Çalıştırırken:

Projeyi eclipse import edin.
Library zip ini açın 
Properties->Java build path add external diyerek dosyaları ekleyin
Properties->Deployment Assembly->Add->Java Build Path Entries burda javax.servlet-api-3.0.1.jar hariç diğerlerini ekleyin.
Run on server komutu ile Tomcat 8 serverini kullanarak çalıştırabilirsiniz.
